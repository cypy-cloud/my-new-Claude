## 1. 회원가입 및 로그인

### 회원가입

1. `/signup` 페이지 접속
2. 이메일·비밀번호 입력 후 **회원가입** 버튼 클릭
3. 가입 즉시 무료 플랜으로 로그인됩니다
4. 첫 로그인 시 서비스 소개 온보딩 튜토리얼이 표시됩니다

### 로그인

1. `/login` 페이지 → 이메일·비밀번호 입력
2. 비밀번호 분실 시: `/forgot-password` → 이메일로 재설정 링크 수신

## 2. AI 문자/카톡 생성 (`/ai-message`)

### 사용 방법

1. **상황 선택**: 가입 안내 / 생일 축하 / 보험금 청구 / 갱신 안내 / 후속 연락 등
2. **고객 이름** 입력 (선택)
3. **추가 요청사항** 입력 (선택)
4. **생성하기** 버튼 클릭

### 결과 활용

- **복사**: 클립보드에 복사해서 카카오톡·문자에 붙여넣기
- **보관함 저장**: 결과물 보관함에 저장
- **재생성**: 마음에 들지 않으면 재생성 버튼 클릭

### 월 사용 한도

| 플랜 | 월 한도 |
|---|---|
| 무료 | 5회 |
| 기본 | 20회 |
| 프로 | 80회 |
| 프리미엄 | 200회 |

## 3. AI 상담 스크립트 생성 (`/ai-script`)

### 사용 방법

1. **상담 목적** 선택 (신규 가입 유치 / 갱신 / 추가 가입 / 클레임 처리 등)
2. **고객 특성** 입력 (나이대·직업·보험 가입 이력 등, 선택)
3. **생성하기** 클릭
4. 10단계 구조의 상담 스크립트 생성

### 결과 구조

1. 도입 (라포 형성)
2. 니즈 파악 질문
3. 현재 상황 분석
4. 문제 제기
5. 솔루션 제안
6. 상품 설명
7. 예상 반론 처리
8. 클로징 질문
9. 계약 체결 유도
10. 후속 조치 안내

## 4. AI PDF 분석 (`/ai-document`)

### 사용 전 주의사항

> **⚠️ 개인정보 포함 PDF 업로드 금지**
> 주민등록번호·연락처·주소·이름 등 개인정보가 포함된 PDF는 업로드하지 마세요.
> 업로드 전 반드시 체크박스에 동의해야 합니다.

### 사용 방법

1. **파일 선택**: PDF 파일 드래그&드롭 또는 클릭 선택
2. 요금제별 파일 크기 제한 확인 (무료: 5MB / 기본: 10MB / 프로: 30MB / 프리미엄: 50MB)
3. **개인정보 없음 확인** 체크박스 동의
4. **업로드 및 분석** 버튼 클릭
5. AI가 PDF 내용을 분석해 쉬운 설명자료 생성

### 결과 활용

- 고객에게 보험 내용 쉽게 설명하는 자료로 활용
- 보관함에 저장 후 나중에 다시 확인 가능

## 5. 블로그·SNS 콘텐츠 생성 (`/content-creator`)

### 사용 방법

1. **주제** 입력 (예: "실손보험 청구 방법")
2. **타깃 고객** 입력 (예: "30~40대 직장인")
3. **보험 분야** 선택
4. **글 길이** 선택 (짧게/보통/길게)
5. **핵심 메시지** 입력
6. **생성하기** 클릭

### 생성 결과

- 블로그 제목 5개
- 블로그 본문 초안
- 인스타그램용 짧은 글
- 페이스북용 글
- 카카오채널용 글
- 해시태그 10~15개

> ⚠️ 실제 게시 전 소속 회사 컴플라이언스 확인 필수

## 6. 뉴스레터 생성 (`/newsletter`)

### 사용 방법

1. **대상 고객층** 입력
2. **주제** 입력
3. **보험 분야** 선택
4. **계절/시기 이슈** 입력 (선택)
5. **사용자 제공 참고 자료** 입력 (AI가 이 내용만 사실로 사용)
6. **생성하기** 클릭

### 생성 결과 (8섹션)

- 제목 / 인사말 / 시사 이슈 1·2·3 / 체크포인트 / CTA / 카카오 요약

### 저장 및 다운로드

- **보관함 저장**: 나중에 `/my-results`에서 다시 확인
- **TXT 다운로드**: 편집 가능한 텍스트 파일로 저장

## 7. 고객 관리 (`/customers`)

### 고객 등록

1. `/customers/new` → 이름·연락처·이메일·생일·직업 입력
2. 보험 가입 정보·메모 추가 가능

### 고객 목록 관리

- 이름/연락처로 검색
- 고객 클릭 → 상세 정보·상호작용 이력 확인
- **일정 관리 버튼** → 해당 고객의 업무 캘린더 연동

### 상호작용 이력

- 상담일·상담 내용·다음 연락 일정 기록
- 문자 발송·전화 등 접촉 이력 관리

## 8. 업무 캘린더 (`/calendar`)

### 일정 추가

1. 오늘/주간/월간 뷰 선택
2. **일정 추가** 버튼 클릭 (또는 월간 뷰에서 날짜 클릭)
3. 제목·일정 유형·날짜·시간·우선순위 입력
4. 고객 연결 (선택) → 고객 관리와 연동

### 일정 유형

| 유형 | 설명 |
|---|---|
| 후속 연락 | 고객 후속 전화·문자 |
| 미팅 | 대면/비대면 상담 |
| 생일 | 고객 생일 알림 |
| 갱신 | 보험 갱신 안내 시기 |
| 검토 | 계약 검토 일정 |
| 기타 | 기타 업무 일정 |

### 완료 처리

일정 카드의 ✅ 버튼 클릭 → `completed` 상태로 변경

## 9. 결과물 보관함 (`/my-results`)

### 기능

- 모든 AI 생성 결과물 목록 조회
- **필터**: 전체 / 문자 / 스크립트 / PDF분석 / 뉴스레터 / 콘텐츠
- **검색**: 내용 키워드 검색
- **다운로드**: 개별 결과물 텍스트 파일 다운로드
- **삭제**: 불필요한 결과물 삭제

### 개인 데이터 백업

1. 보관함 우측 상단 **"내 데이터 백업"** 버튼 클릭
2. 전체 결과물 + 원본 PDF + 사용량 기록이 ZIP으로 다운로드

## 10. 요금제 업그레이드 (`/billing`)

1. 사이드바 → **요금제** 메뉴
2. 원하는 플랜의 **업그레이드** 버튼 클릭
3. 결제 진행 (Toss Payments 연동 후 활성화 예정)

### 한도 초과 시

사용 한도 초과 시 오류 메시지가 표시됩니다:
> "이번 달 사용 한도(N회)를 초과했습니다. 플랜을 업그레이드해주세요."

## 11. 계정 설정 (`/settings`)

- **이름 변경**: 사이드바에 표시되는 이름
- **비밀번호 변경**
- **알림 설정**: 이메일 알림 수신 여부
- **계정 탈퇴**: 회원 탈퇴 (데이터 완전 삭제 요청)
